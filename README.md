# 🌌 Glassmorphic Developer Portfolio

A premium, modern, and highly interactive single-page portfolio built for developers. This project features a clean dark mode pekat, high-end glassmorphism design effects, dynamic micro-interactions, and ultra-smooth performance.

---

## 🚀 Tech Stack

- **Frontend core:** HTML5, Vanilla JavaScript
- **Styling framework:** Tailwind CSS (Utility-first framework with custom glassmorphism components)
- **Smooth scrolling:** Lenis Scroll CDN (Premium Apple/Vercel-style scroll animation)
- **Iconography:** Lucide Icons / Font Awesome CDN

---

## ✨ Features & Gimmicks

- **Glassmorphic Aesthetic:** Translucent components with sophisticated blur effects (backdrop-blur-md) and glowing borders tailored for both dark and light variations.
- **Dual-Theme Toggle:** Seamless real-time switching between a deep dark setup (bg-slate-950) and a crisp, modern off-white light layout (bg-slate-50).
- **Vercel-Style Live Navigation:** Floating context background indicator that tracks the cursor position dynamically across navbar items and snaps back to the active viewport section on exit.
- **Interactive Custom Cursor:** Hardware-accelerated fluid dot and floating cursor outline loop tracking system for desktop browsers (automatically adapts/disables on touch screens).
- **Project Grid Showcase:** Dynamic presentation cards integrated directly with real project data and responsive target anchors matching active source repositories.

---

## 🛠️ Included Showcases

1. **Pustalora** — Digital Book Lending System built using Laravel, MySQL, and Tailwind CSS.
2. **Travel Jogja** — Regional tourism information system designed to provide interactive local destination discovery.
3. **Web Statistics** — Live telemetry monitoring and visualization dashboard template utilizing Node.js & Express.

## 💻 Local Installation & Setup

To view and run this repository locally on your computer, simply follow these steps:

1. **Clone the repository:**
git clone https://github.com/RasyidRlqr/My-portfolio.git

2. **Navigate into the directory:**
cd My-portfolio

3. **Launch the interface:**
Since this portfolio runs completely via raw CDN compilation and light Vanilla execution, you don't need to run any heavy build servers or npm parameters. Simply click and run the index.html file inside any standard web browser or launch it natively through the Live Server extension in VS Code.
